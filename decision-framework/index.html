Cost Estimation Grid

Approximate monthly costs for a typical 3-tier application (web + app + database)

Backup & Restore

~$50–200/mo
  • S3 backup storage (cross-region)
  • EBS snapshot storage
  • AWS Backup service (no charge)
  • IaC templates (no charge)
  • No running compute in DR region

Pilot Light

~$200–800/mo
  • DRS replication ($0.028/server/hr)
  • Staging area EBS volumes
  • RDS Cross-Region Read Replica
  • AMI/snapshot storage
  • Compute ON DEMAND only during DR

Warm Standby

~$1,000–5,000/mo
  • Scaled-down EC2 fleet (always running)
  • Aurora Global Database replica
  • ARC cluster ($2.50/hr = ~$1,800/mo)
  • Load balancer in DR region
  • DRS for non-DB servers

Active-Active (Multi-Site)

~$5,000–20,000+/mo
  • Full compute fleet in 2+ Regions
  • DynamoDB Global Tables / Aurora Global
  • ARC + Global Accelerator
  • Dual ALBs + Auto Scaling Groups
  • ~2× production costs
⚠️ Note: Costs are illustrative estimates for a mid-size 3-tier app. Actual costs depend on data volume, instance sizes, transfer, and region pricing. Always use the AWS Pricing Calculator for accurate estimates.
